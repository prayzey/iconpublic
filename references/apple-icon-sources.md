# Apple Sources

Use these primary references when verifying icon behavior:

- CFBundle icon key (`CFBundleIconFile`):
  https://developer.apple.com/documentation/bundleresources/information_property_list/cfbundleiconfile
- NSApplication icon API (`applicationIconImage`):
  https://developer.apple.com/documentation/appkit/nsapplication/1428671-applicationiconimage
- Apple QA on iconset/icns tooling (`iconutil`):
  https://developer.apple.com/library/archive/qa/qa1686/_index.html
- SwiftPM resource loading (`Bundle.module`):
  https://github.com/swiftlang/swift-package-manager/blob/main/Sources/PackageManagerDocs/Documentation.docc/BundlingResources.md
